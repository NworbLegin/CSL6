Detailed Description

Examples.

– Simple LPF (2 bands) at 0.2 Fs with 0.05-width transition bands responses = { 1 x 0 }; freqs = { 0 0.2 0.25 0.5 }; weights = { 10 20 };

– basic BPF (3 bands) between 0.2 and 0.3 Fs with 0.05-width transition bands responses = { 0.5 x 1 x 0.8 }; freqs = { 0 0.15 0.2 0.3 0.35 0.5 }; weights = { 20 5 20 };

– Fancier dual-BPF responses = { 0 x 1 x 0 x 1 x 0 }; freqs = { 0 0.05 0.1 0.15 0.18 0.25 0.3 0.36 0.41 0.5 }; weights = { 10 1 3 1 20 }; FIR Filter class
