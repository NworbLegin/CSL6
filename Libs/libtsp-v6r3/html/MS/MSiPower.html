MSiPower

Routine

int MSiPower (int a, int n)

Purpose

Integer raised to an integer power

Description

This routine calculates a raised to the n'th power. Zero raised to the zero'th power is returned as one.

Parameters

<- int MSiPower
Returned integer value
-> int a
Input value
-> int n
Integer exponent
