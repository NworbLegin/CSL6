MSdNint

Routine

double MSdNint (double x)

Purpose

Nearest integer function

Description

This routine returns the integer-valued double nearest x. Specifically, the returned value is floor(x+0.5) for positive x and ceil(x-0.5) for negative x.

Parameters

<- double MSdNint
Returned integer value
-> double x
Input value
