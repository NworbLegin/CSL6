VRlPrint

Routine

void VRlPrint (FILE *fp, const char Header[], const long int x[], int N)

Purpose

Print a long integer array along with a header text

Description

This routine prints the values of a long int array along with an identifying header.

Parameters

-> FILE *fp
File pointer for the output. If fp is NULL, no printing occurs.
-> const char Header[]
Character string for the header. The header line is omitted if the header string is empty.
-> const long int x[]
Array of N long ints
-> int N
Number of elements in x
