VRfSet

Routine

void VRfSet (double a, float y[], int N)

Purpose

Set each element in an array of floats to a given value

Description

This routine sets each element in a given float vector to the value a, 
  y[i] = a,  0 <= i < N.

Parameters

-> double a
Value to be set
<- float y[]
Array of floats (N elements)
-> int N
Number of elements
