VRfPow10

Routine

void VRfPow10 (const float x[], float y[], int N)

Purpose

Find 10 raised to the power of each element of an array

Description

This routine finds the anti-logarithm (base 10) for each element of an array. For each element in the input array, the output is given as 
  y[i] = 10 ^ (x[i]),  0 <= i < N.

Parameters

-> const float x[]
Input array (N elements)
<- float y[]
Output array (N elements). The output array can be the same as the input array.
-> int N
Number of elements in the arrays (may be zero)
