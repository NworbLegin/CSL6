VRfPrint

Routine

void VRfPrint (FILE *fp, const char Header[], const float x[], int N)

Purpose

Print a float array along with a header text

Description

This routine prints the values of a float array along with an identifying header.

Parameters

-> FILE *fp
File pointer for the output. If fp is NULL, no printing occurs.
-> const char Header[]
Character string for the header. The header line is omitted if the header string is empty.
-> const float x[]
Array of N floats
-> int N
Number of elements in x
