VRfDotProd

Routine

double VRfDotProd (const float x1[], const float x2[], int N)

Purpose

Dot product of two float arrays (double result)

Description

This routine calculates the dot product of two float arrays x1 and x2. The dot product is calculated internally as a double and returned as a double value. 
               N-1
  VRfDotProd = SUM x1[i] * x2[i] .
               i=0

Parameters

<- double VRfDotProd
Dot product
-> const float x1[]
Input array (N elements)
-> const float x2[]
Input array (N elements)
-> int N
Number of elements in the arrays (may be zero)
