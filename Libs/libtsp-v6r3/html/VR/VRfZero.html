VRfZero

Routine

void VRfZero (float y[], int N)

Purpose

Zero an array of floats

Description

This routine zeros N elements of a given float array, 
  y[i] = 0.0,  0 <= i < N.

Parameters

<- float y[]
Array of floats (N elements)
-> int N
Number of elements
