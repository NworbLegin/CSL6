VRiZero

Routine

void VRiZero (int y[], int N)

Purpose

Zero an array of ints

Description

This routine zeros N elements of a given int array, 
  y[i] = 0,  0 <= i < N.

Parameters

<- int y[]
Array of ints (N elements)
-> int N
Number of elements
