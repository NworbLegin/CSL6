VRdZero

Routine

void VRdZero (double y[], int N)

Purpose

Zero an array of doubles

Description

This routine zeros N elements of a given double array, 
  y[i] = 0.0,  0 <= i < N.

Parameters

<- double y[]
Array of doubles (N elements)
-> int N
Number of elements
