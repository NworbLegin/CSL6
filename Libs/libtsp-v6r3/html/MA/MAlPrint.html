MAlPrint

Routine

void MAlPrint (FILE *fp, const char Header[], const long int *A[], int Nrow, int Ncol)

Purpose

Print a long integer matrix along with a header text

Description

This routine prints the values of a long int matrix along with an identifying header. No printing occurs if the file pointer is NULL. The values for each row of the matrix are printed 5 to a line, with each row starting on a new line.

Parameters

-> FILE *fp
File pointer for the output. If fp is NULL, no printing occurs.
-> const char Header[]
Character string for the header. This routine supplies a terminating newline. The header line is omitted if the header string is empty.
-> const long int *A[]
Pointer to an array of row pointers. A is interpreted as an N by M matrix. Note that with ANSI C, if the actual parameter is not declared to have the const attribute, an explicit cast to (const long int **) is required.
-> int Nrow
Number of rows in the matrix
-> int Ncol
Number of elements in each row (number of columns)
