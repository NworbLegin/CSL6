UTfree

Routine

void UTfree (void *ptr)

Purpose

Deallocate a block of memory

Description

This routine deallocates a block of memory allocated by UTmalloc (or malloc).

Parameters

-> void *ptr
Pointer to the memory to be deallocated. If ptr is NULL, no action is taken.
