FNsinc

Routine

double FNsinc (double x)

Purpose

Evaluate the sin(pi*x)/(pi*x) function

Description

This function returns the value of the sinc function, where 
             sin (pi x)
  sinc (x) = ---------- .
                pi x

Parameters

<- double FNsinc
Returned value
-> double x
Input value
