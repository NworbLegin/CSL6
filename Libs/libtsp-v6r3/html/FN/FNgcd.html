FNgcd

Routine

long int FNgcd (long int u, long int v)

Purpose

Find the greatest common divisor of two integer values

Description

This function finds the (positive) greatest common divisor of two integer values. If both elements are zero, the greatest common divisor is set to zero.

Parameters

<- long int FNgcd
Greatest common divisor of u and v
-> long int u
First integer value
-> long int v
Second integer value
